== Description ==
This plugin activates a honeypot (Anti-Spam and anti-bot) in the following sites:
-WooCommerce login form
-WooCommerce registration form
-Comments box of the post
-WordPress registration form
-WordPress login form

It is highly optimized, so that the performance of your website is not affected

== Installation ==
1. Download the repository plugin
2. Activate the plugin from the WordPress menu
